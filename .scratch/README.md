# .scratch/ — 本地工作项

这个目录用于 **local markdown 风格的工作项追踪**（见 [docs/workflow.md](../docs/workflow.md)）。

## 用法

- 每个功能/主题建一个子目录，如 `.scratch/add-export-feature/`。
- 每个目录里放 markdown 文件描述任务、想法、待办。
- 这些是**本地工作草稿**，不进对外文档，可随时清理。

## 与 `/to-prd`、`/grill-me` 的关系

- `/grill-me` 的输出可以记到这里（拷问后的想法）。
- `/to-prd` 产出的 PRD 也先放这里，确认后再提升到正式位置。
